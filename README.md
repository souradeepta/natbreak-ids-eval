# NATbreak — Paper 09

**"NATbreak: Exploiting Enterprise Network Address Translation as a Zero-Cost Adversarial Amplifier for ML-Based Intrusion Detection"**

Target venue: IEEE TIFS / S&P

## Quick start

```bash
pip install -r requirements.txt
make all          # runs experiments + tests + compiles paper
make test         # 63 unit tests
make experiments  # generates results/macros09.tex + 6 figures
```

## Structure

```
src/
  natbreak_model.py   # NAT amplification model, adversarial IDS attack
  strategies.py       # defense strategies, invariant analysis
  experiments.py      # compute results, write macros09.tex, dispatch figures
  figures/
    fig1_baseline_vs_nat.pdf
    fig2_accuracy_vs_nat_ratio.pdf
    fig3_entropy_collapse.pdf
    fig4_attack_by_variant.pdf
    fig5_invariant_comparison.pdf
    fig6_amplification.pdf
tests/
  test_natbreak.py    # 63 tests covering model, attacks, defenses
paper/
  paper09.tex         # IEEEtran, conference mode
  paper09.bib         # references
results/
  macros09.tex        # macros (auto-generated by experiments.py)
```
