# Paper 09: natbreak-ids-eval — Developer Explainer

## What This Paper Does

This paper is part of the EB2-NIW IEEE cybersecurity portfolio (Paper 09 of 10).
It provides original research in the natbreak-ids-eval domain with formal mathematical
analysis, empirical evaluation, and practical implementation.

## Repo Structure

```
src/         # Core implementation
experiments/ # Runs evaluation, writes results/results.tex
tests/       # Unit tests (all passing)
paper/       # IEEE LaTeX paper
results/     # Auto-generated figures and macro file
docs/        # This directory
```

## Running

```bash
pip install -r requirements.txt
python3 -m pytest tests/ -v
cd paper && pdflatex paper09.tex && bibtex paper09 && pdflatex paper09.tex && pdflatex paper09.tex
```
